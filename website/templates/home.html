Error Code Logging Tool

This application provides an interface for a user to search and create
error logs that correspond with machine generated error codes. I created
it with the a horizontal expansion machine in mind.

I operated one of these machines to expand radiator coils. The process of
looking up the error codes in the printed book and reading the other
operators handwritten logs was tedius.

I used the pdf of the manual and a python script to pull the individual error
codes out for the datatbase. Originally I created the program with tkinter
and python. My goal with this project was to place the application in a state
that was more easily distributable using Python and Flask.

This app is created with Flask and SQL-Alchemy. To use the app go to the
login page and enter the guest username provided at the top of the screen.
